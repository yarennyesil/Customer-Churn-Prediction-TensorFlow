# Müşteri Kaybı (Churn) Tahmini - TensorFlow

Bu proje, Python ve TensorFlow kullanarak bir şirketin müşterilerinin ayrılıp ayrılmayacağını tahmin eden bir derin öğrenme modelidir.

##  Sonuçlar
- **Doğruluk Oranı:** %96.67
- **Model Yapısı:** 2 Gizli Katmanlı Yapay Sinir Ağı (ANN)
- **Kullanılan Teknolojiler:** TensorFlow, NumPy, Matplotlib

## Nasıl Çalıştırılır?
1. Bilgisayarınızda Python ve TensorFlow yüklü olduğundan emin olun.
2. `.ipynb` dosyasını VS Code veya Jupyter Notebook ile açın.
3. Tüm hücreleri çalıştırın.
